Class: Chart

Chart(margin, width, height, data, keydescriptiontext, valuedescriptiontext)

A class called Chart is used to create the visulisation.

Constructor

new Chart(margin, width, height, data, keydescriptiontext, valuedescriptiontext)

Parameters:
Name Type Default Description
margin number 40 The margin value of the visulisation viewbox.
width number 300 The width value of the visulisation.
height number 300 The height value of the visulisation.
data The data value which is parsed from parsedata.
keydescriptiontext string Country: The text which describes the key data used on the first line when hovering over visulisation.
valuedescriptiontext string GDP of Country: $ The text which describes the value data used on the first line when hovering over visulisation.

Example
var chart = new Chart(40, 300, 300, data, 0.0, 'Country: ', 'GDP of Country: $');
